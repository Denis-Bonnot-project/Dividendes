# Dividendes  

## **Descriptif**
 
L'application trace l'évolution des dividendes (ajustés) des actions du CAC 40.


#### Notes

* Si l'action a été divisée, les dividendes précédents sont divisés de la même manière et constituent alors les _dividendes ajustés_.
* Les données on été téléchargées via l'API de "Financial Modeling Prep".
